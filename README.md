# Superstore Sales Analysis & Dashboard

### Project Overview
This project is an end-to-end data analysis of the Superstore dataset. The goal was to clean, analyze, and visualize sales data to identify key business insights.

The process involved:
1.  **Data Cleaning:** Handled missing values, corrected data types, and checked for duplicates using Python (Pandas).
2.  **Exploratory Data Analysis (EDA):** Used Python (Seaborn/Matplotlib) to analyze sales trends over time, identify top-performing categories and regions, and find the most valuable customer segments.
3.  **Data Visualization:** Created an interactive dashboard in Google Looker Studio to present the findings.

### Tools Used
* **Python:** (Pandas, Matplotlib, Seaborn)
* **Jupyter Notebook:** For analysis and code
* **Google Looker Studio:** For dashboarding
* **Git & GitHub:** For version control
